# Aurora Gem - full web stack app

This repo is for laboratory classes WDAI in AGH
Main used technolgies in project are : React, express, TS, Material UI

# Setting up
```sh
cd /backend

#installing depedencies
npm install

#running backend
npm run dev

cd ../frontend

npm install

npm run dev

echo "Good to go :tada:"
```
